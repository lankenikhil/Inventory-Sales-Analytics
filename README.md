## Problem Statement

Retail and wholesale organizations often struggle with:

- Excess inventory
- Vendor dependency
- Low-performing products
- Pricing inefficiencies
- Reduced profitability

This project applies data analytics techniques to identify these issues and provide actionable business recommendations.

## Project Workflow

### 1. Data Collection

Collected inventory, sales, and vendor datasets.

### 2. Data Cleaning

- Removed missing values
- Removed invalid records
- Filtered loss-making transactions

### 3. Exploratory Data Analysis

Performed:

- Summary Statistics
- Correlation Analysis
- Profitability Analysis
- Inventory Analysis

### 4. Vendor Analysis

Identified:

- Top Revenue Vendors
- Low Performing Vendors
- Vendor Dependency Risks

### 5. Inventory Analysis

Evaluated:

- Stock Turnover
- Unsold Inventory
- Purchase Optimization

### 6. Dashboard Development

Built interactive dashboards using:

- Power BI
- Tableau

---

## Key Findings

- Top 10 vendors contribute 65.69% of total purchases.
- 198 brands show low sales but high profit margins.
- Bulk purchasing reduces unit costs significantly.
- Unsold inventory capital exceeds $2.71M.
- Significant differences exist between top and low-performing vendors.

---

## Business Recommendations

- Diversify vendor relationships.
- Optimize pricing strategies.
- Improve inventory turnover.
- Reduce slow-moving stock.
- Enhance marketing for high-margin products.
